-datatypes :tells the computer what kind of information you are working with
2 types:
   -Primitive datatypes : they store simple values
                  -Number: any number ex:11,22,23.5,234.89

 -operators in JS 
    1.Assignment Operators(=): used to assign values
            variable_name=value
    2.Arithmetic Operators: used to perform math operations
            + addition
            - substraction
            * multiplication
            / division
            % modulus
            ** exponentiation
    3.Relational(comparison) Operators: used to compare values(return true or false)
             == equal
             === strict equal
             != not equal
             > greater than
             < less than
             >= greater than or equal
             <= less than or equal
     4.Logical Operators:
             && and
             || or
             ! not

     ---------and---------------or------------not---------
          
          T && T = T       T || T = T         F!=T 
          T && F = F       T || F = T         T!=F
          F && T = F       F || T = T 
          F && F = F       F || F = F 

     5.Unary Operators:works on a single operand
            ++ increment
            -- decrement

     6.Shorthand Operators:(+=,-=,*=,/=)

     7. Truthy Operator(&&)
           syntax:condition && Statement
     
     8.Ternary Operator(?:)
       syntax:condition ? true statement : false statement

     9.Spread Operator(...) : used to expand(spread out) elements of an array,
       object or iterable into individual element

     10.Rest Operator(...) : used to collect multiple values into a single variable
        (usually as an array)

                        Syntax:

                                   Statements

- Array:
        An array is a data structure in js that 
        allows us to store multiple values of 
        any data type in a single variables

        -syntax: variable_name=[] / variable_name=["item1","item2","item3"....]
        -type of an array is an "OBJECT"
        -index : position number used to identify and access elements in an array
        -accessing an array element : array_name[index]
        -to find total elements stored in an array
            syntax: array_name.length
        -to access array elements
            -using for-loop
            -using for-of loop
            -using for-in loop
       - Array Methods : pre-defined methods to manipulate array data
            -push(item) : used to insert data at the end of an array
            -unshift(item) : used to insert data at the first of an array
            -pop() : used to remove last element of array
            -shift(): used to remove the first element of an array
            -foreach : used to loop through each element
               syntax:
                   foreach((element,index,array)=>console.log(element))

                   -element-current value in the array
                   -index-current index(optional)
                   -array-original array(optional)

            -filter : used to create a new array with elements that pass
                      a certain condition
              syntax:
                   filter((element,index,array)=>condition)

            -find() : used to return the first element in 
                      an array that satisfies a condition
                     syntax:
                           find((element,index,array)=>condition)

            -map() : used to transform each elements of an array and
                     return new array 
                     syntax:
                           map((element,index,array)=>newvalue) 

            -reduce() : method is used to reduce an array to a single value.
                        it return either smallest/largest/total sum of elements
                        in an array 
                        syntax: 
                               array.reduce((accumalator))

            -some() : used to check whether atleast one element in an array satisfies
                      a given condition (return boolean value)   

            -flat() : used to flatten a nested array into a single-level array
                     syntax:
                            array.flat(depth)
                     depth : optional,specify how deep the nested array should be flattened
                     default value=1                                
            
            -includes(item) : used to check whether an array contains a specific value,return boolean value
                      syntax: array.include(value,startIndex)
                          value: value to search
                          startIndex: optional(default=0)

            -indexOf() : return the index position of a value in an array
                     syntax: array.indexOf(value,startIndex)
                          value: value to search
                          startIndex: optional(default=0)
                             
             -splice() : used to add elements,remove elements and replace elements
                         from the existing array .(it modifies the original array) 
                     syntax: array.splice(startIndex,deleteCount,item1,item2,.,.,.,.,)

                      -startIndex: Where to start
                      -deleteCount: how many element to remove            
                      -item1,item2...-elements to insert(optional)
              
             -join() : converts an array into a string by joining all elements
                     syntax: array.join(seperator)

                       -seperator: optional(default value is comma(,))

OOP (Object Oriented Programming)
---------------------------------

OOP is a programming approach where we create programs using object that contains
       .data(properties)
       .functions(methods)

it models real-world entities(like student,car,bank account) as object in software

 -class: blueprint defining structure and behaviour of objects
 -object: runtime instance of a class withstate and behaviour
 -reference: 'this' keyword:refers to the current object that is executing the function.
          it is used to access properties and methods belonging to the same object.
 -constructor: is a special method in a class that is automatically called when an object is created.
               it is mainly used to initialize object properties.

-OOP Features

     -inheritance:Acquirng properties and methods of one class into another class
                  -inheritance btwn classes is known as "classical inheritance"
                  -inheritance btwn objects is known as "prototype inheritance"

     -polymorphism : one method can perform different taska based on the object.simply,same name different behaviour

     -encapsulation : binding data(variables),and methods(functions) together into a single unit and restricting direct access
                      to some data.simply,
                      = data hiding + wrapping into one unit = encapsulation.
     -abstraction : hiding internal implementation details and showing only the necessary features to the user

-inorder to give behaviour to your webpages
-ways to apply js in web programmers
      -internal js : use <script> tag to provide js code in html
      -external js : link external js file with HTML using <script> tag

-DOM(Document Object Model)
  
   it is a tree like structure representation of HTML Document that allows js to access,modify,add,or 
   delete HTML elements of a webpage dynamically

      when a page loads in the browser

           -browser reads the html Document
           -converts into a tree-like structure(DOM)
           -each html tag becomes an object(node)
           -js can access these object using DOM

       -selecting html elements into js code

          -using tag name : document.getElementsByTagName('tagname')
          -using ID : document.getElementById('id name')
          -using class : document.getElementByClassName('class')
          -selects only the first matching element : document.querySelector('tagname'/'#id'/'.class')
          -selects all matching element : document.querySelectorAll('tagname'/'#id'/'.class')

       -DOM Methods
          -createElement()-create new HTML element
          -textContent/innerHTML/innerText-adds contents
          -appendChild-adds element to webpage
          -remove()-to remove element from webpage

-Events : an event in js is an action or occurance that happens in the 
          browser,which js can detect and respond to

          user action                 Event name
         ----------------------------------------
          click button                  click
          move mouse                    mouseover
          press keyboard                keydown
          submit form                   submit
          load page                     load

       -ways to handle events in js

        -inline eventhandling (HTML elements)
        -using DOM property
        -using addEventListener()
               syntax:element.addEventListener("event_name",event_handler)

        -Event capturing and event bubbling are mechanisms of event propagation in
         the DOM that controls the order in which event handlers are executed from
         parent to child or child to parent
